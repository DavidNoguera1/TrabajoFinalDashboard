# TrabajoFinalDashboard

Comandos exactos para correr backend y frontend.

## Backend (UMariana DWH)

```bash
cd Backend\umariana_dwh-backend
npm install
npm run dev
```

## Frontend (Dashboard)

```bash
cd Frontend\dashboard-dei
npm install
npm run dev
```

## Nota

Ejecuta backend y frontend en terminales separadas.
